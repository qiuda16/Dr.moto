# 维修手册标准采集模板

更新时间：2026-04-06

## 目标
所有维修手册、扭矩表、保养标准、线路图在进入系统后，统一按同一套结构抽取与补录，保证车型资料、维修项目、工单引用、后续 Jarvis 检索使用同一口径。

## 一级模板
1. 资料基本信息
   - 文档标题
   - 文档类型
   - 语言
   - 页数
2. 适用车型范围
   - 品牌
   - 车型
   - 年款范围
   - 发动机型号
3. 维修/保养项目索引
   - 项目名称列表
   - 章节索引
4. 标准施工步骤
   - 标准步骤
   - 所需工具
   - 安全提示
   - 质检点
5. 关键技术参数
   - 扭矩参数
   - 油液/容量参数
   - 胎压参数
   - 电压参数
   - 间隙/公差参数
6. 配件与耗材
   - 零件清单
   - 耗材
   - 油液
7. 可追溯信息
   - 来源页码
   - 页面摘要/原文摘录

## 入库原则
1. 能识别的字段直接结构化保存。
2. 识别不到的字段保留为空，后续人工补录。
3. 高风险参数不能仅靠模型自由生成，必须保留来源页码。
4. 施工步骤优先挂到车型维修手册条目。
5. 后续维修项目、工时、配件关联，都以这个模板为标准源头继续扩展。

## 当前系统实现
1. OCR 解析结果会输出 `manual_template`。
2. OCR 解析结果会输出 `normalized_manual`。
3. 前端资料库解析结果页会展示模板完成度、缺失字段和标准化结构。
4. 一键导入维修手册条目时，优先按标准化模板写入。
