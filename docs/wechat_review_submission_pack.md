# 微信小程序提审资料包（可直接粘贴）

## 1. 小程序名称

车主服务中心

## 2. 小程序简介（80字以内）

为车主提供车辆体检信息、保养记录、推荐保养项目与养护知识，帮助客户更清晰地了解车辆状态并做出维护决策。

## 3. 主要功能说明

1. 车辆总览：首页展示关键状态（里程、体检次数、最近工单状态）
2. 体检记录：查看历史体检数据与关键检测项
3. 保养记录：查看工单状态、日期与金额
4. 推荐保养：按优先级给出建议项目与参考价格
5. 养护科普：查看车辆养护资料
6. 个人中心：查看账户信息与门店联系方式

## 4. 用户体系说明

- 用户通过微信授权登录
- 首次登录需绑定手机号与车牌
- 用户仅可查看本人车辆相关数据

## 6. 数据与隐私说明（审核说明）

- 采集信息：微信 openid、手机号、车牌号、车辆体检与保养数据
- 使用目的：用于用户身份识别、车辆数据查询与保养服务
- 不会向无关第三方出售用户个人信息
- 用户可在“我的”页面查看服务联系方式并申请服务支持
